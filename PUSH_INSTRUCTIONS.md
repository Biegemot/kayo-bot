```bash

git remote set-url origin git@github.com:Biegemot/kayo-bot.git
git push origin main
```

#### Вариант B: С токеном доступа
```bash
# Создайте токен в GitHub: Settings → Developer settings → Personal access tokens
# Дайте разрешения: repo (полный контроль)
git remote set-url origin https://github.com/Biegemot/kayo-bot.git
git push origin main
# При запросе логина: используйте свой username
# При запросе пароля: введите токен доступа
```

3. **Убедитесь, что есть доступ** к репозиторию

### ЕСЛИ PUSH НЕ РАБОТАЕТ
1. **Создайте форк** репозитория
2. **Добавьте remote** для форка:
   ```bash
   git remote add fork https://github.com/YOUR_USERNAME/kayo-bot.git
   git push fork main
   ```
3. **Создайте Pull Request** через веб-интерфейс GitHub
