## 🚀 ИНСТРУКЦИИ ПО ИСПОЛЬЗОВАНИЮ

### 📝 **Использование системы логирования**
```python
from bot.logging import logger, log_function_call

# Базовое логирование
logger.info("Сообщение", user_id=123, action="command")

# Логирование с контекстом
logger.set_context(component="webapp", session_id="abc123")
logger.debug("Отладка компонента")

# Декоратор для автоматического логирования
@log_function_call
def my_function(param):
    return param * 2
```

### 🛠️ **Запуск диагностики**
```python
from bot.diagnostics import run_diagnostics
import asyncio

# Запуск всех тестов
report = asyncio.run(run_diagnostics())
print(f"Результаты: {report['summary']}")
```
