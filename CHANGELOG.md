# Changelog

Все заметные изменения в проекте будут документироваться в этом файле.

Формат основан на [Keep a Changelog](https://keepachangelog.com/ru/1.0.0/),
и этот проект придерживается [Semantic Versioning](https://semver.org/lang/ru/).

## [0.8.1] - 2026-04-03

### Добавлено
- Windows GUI интерфейс с системным треем
- Linux установщик для systemd службы
- Docker контейнеризация
- Автоматическая сборка релизов через GitHub Actions
- Система анкет с inline-редактированием
- Команда `/me edit` для редактирования анкет

### Изменено
- Улучшена совместимость с Windows
- Оптимизирована работа с базами данных per-чат
- Улучшена система логирования
- Обновлена документация README.md

### Исправлено
- Критические ошибки в ActivityManager
- Проблемы с импортами модулей
- Ошибки обработки версий

## [0.8.0] - 2026-04-02

### Добавлено
- Полная система анкет для фурсон
- Inline-редактирование анкет через личный чат
- Поддержка фото-референсов
- Расширенные поля анкет (вид, возраст, ориентация, город и т.д.)
- Команда `/me @username` для просмотра анкет других пользователей

### Изменено
- Переработана структура проекта
- Улучшена система импортов
- Оптимизирована работа с БД

### Исправлено
- Ошибки в системе обновлений
- Проблемы с обработкой команд
- Утечки памяти в ActivityManager

## [0.7.0] - 2026-03-30

### Добавлено
- Команда `/summarize` для ежедневного отчёта активности чата
- Команда `/titles` для просмотра доступных титулов
- Автообновление бота при запуске и каждые 6 часов
- Система титулов на основе активности пользователя
- Поддержка per-чат баз данных SQLite

### Изменено
- Улучшена документация README.md
- Исправлены опечатки в документации
- Улучшена обработка версий в команде `/about`

### Исправлено
- Дублирование docstring в `bot/handlers/rp_utils.py`
- Некорректное упоминание GUI в README.md

## [0.6.0] - 2026-03-XX

### Добавлено
- Команды RP: `/hug`, `/bite`, `/pat`, `/boop`, `/kiss`, `/slapass`
- Автоматические реакции на сообщения
- Отслеживание активности пользователей
- Команды статистики: `/top`, `/today`, `/me`

### Изменено
- Первый стабильный релиз бота

## [0.5.0] - 2026-03-XX

### Добавлено
- Базовая структура бота
- Поддержка конфигурации через `.env`
- Установщики для Windows и Linux

---

**Формат изменений:**
- `Добавлено` — новые возможности
- `Изменено` — изменения в существующей функциональности
- `Удалено` — удалённые возможности
- `Исправлено` — исправления ошибок
- `Безопасность` — изменения безопасности

**Версионирование:**
- `MAJOR.MINOR.PATCH`
- **MAJOR** — несовместимые изменения API
- **MINOR** — новая функциональность с обратной совместимостью  
- **PATCH** — исправления ошибок
